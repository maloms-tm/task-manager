# Task Manager Application

A modern task management application built with Angular 17.

## Features

- Create, edit, and delete tasks
- Filter tasks by priority and status
- Persistent storage using LocalStorage
- Responsive design
- Form validation
- Comprehensive unit tests

## Tech Stack

- Angular 17
- TypeScript 5.2
- RxJS 7.8
- Angular Material 17
- Jasmine & Karma

## Setup

```bash
npm install
npm start
```

Open `http://localhost:4200`

## Testing

```bash
npm test
```

## Build

```bash
npm run build
```
